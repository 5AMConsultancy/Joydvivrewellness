# Photos go here

Drop the real photos into this folder with these exact filenames and they will
appear on the site automatically — no code changes needed:

| Filename        | Where it appears                        |
|-----------------|------------------------------------------|
| `logo.jpg`      | Logo — header (top left) and hero circle (already added) |
| `about.jpg`     | Photo of Liesl in the About section (already added) |
| `gallery-1.jpg` | Gallery grid, position 1                 |
| `gallery-2.jpg` | Gallery grid, position 2                 |
| `gallery-3.jpg` | Gallery grid, position 3                 |
| `gallery-4.jpg` | Gallery grid, position 4                 |
| `gallery-5.jpg` | Gallery grid, position 5                 |
| `gallery-6.jpg` | Gallery grid, position 6                 |
| `gallery-7.jpg` | Gallery grid, position 7                 |
| `gallery-8.jpg` | Gallery grid, position 8                 |
| `gallery-9.jpg` | Gallery grid, position 9                 |

Until a photo is added, the site shows an elegant turquoise placeholder in its place.

Tips:
- JPG format, ideally under ~500 KB each for fast loading.
- Gallery images are displayed as squares (cropped to fit), so roughly square
  photos look best.
